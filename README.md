# ShowMark

## About

Markdown component with [Showdown.js](https://github.com/showdownjs/showdown) and [@types/mdx](https://www.npmjs.com/package/@types/mdx).

## Install

```bash
npm i showmark
```

```bash
pnpm i showmark
```

```bash
yarn add showmark
```

## Usage

`example.md`

```markdown
---
title: "Hello"
date: "2024-12-12"
---

# Hello World
```
